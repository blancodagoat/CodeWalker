CodeWalker is a viewer and editor for Grand Theft Auto V (GTAV) game files. It provides tools to explore, visualize, and modify the game's RPF archives and various file formats.

Key capabilities:
- 3D world viewer with camera controls and entity selection
- RPF archive explorer for browsing game files
- Map editor for creating and modifying YMAP files (entity placements)
- Traffic path (YND) and scenario (YMT) editing
- Texture, model, and audio file viewers
- Project-based workflow for creating map mods
- Multiple launch modes: world view, menu mode, and explorer mode

The application is designed for modders and developers who want to understand or modify GTAV's game content, including 3D models, textures, collision data, and world placement.
